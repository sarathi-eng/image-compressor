
## 2024-05-24 - Contextual Tooltips for Disabled States
**Learning:** Adding helpful `title` attributes (tooltips) to disabled buttons significantly improves the user experience by explaining *why* an action is currently unavailable (e.g., "No images to compress" vs just a grayed-out button). Without them, users can be confused about how to unlock functionality.
**Action:** Always add contextual `title` (or ARIA descriptions) to disabled UI elements to guide users on the necessary preconditions for the action.
