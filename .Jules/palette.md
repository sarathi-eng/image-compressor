## 2024-06-25 - Context for disabled buttons
**Learning:** Adding descriptive `title` attributes (or aria descriptions) to disabled UI elements provides crucial context for users, explaining *why* an action is currently unavailable. This is especially helpful for users with cognitive disabilities or those relying on screen readers, as well as general users who might be confused about a button's inactive state.
**Action:** Always provide context for disabled UI elements by adding descriptive `title` attributes or ARIA descriptions to explain why the action is currently unavailable.
